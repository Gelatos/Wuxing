Character Creation
This is the Character Creator. Here you can set starting aspects of your character like their origins, starting ability scores, and starting skills. You can always return to this page from Options. Submit will have you accept any changes made on this page and add your selections to your character. You will then be sent to Class Advancement to select class(es) to level up. Leave will ignore any changes made here and return you to the Character Sheet.
